Commands

  • -session start
    Description: Sends your SSU message in the SSU channel you provided in the setup.
    Usage: -session start
  • -session stop
    Description: Sends your shutdown/SSD message in the channel you provided in the setup.
    Usage: -session stop
  • -session full
    Description: Used for when your roleplay server is full. Sends a session full embed.
    Usage: -session full
  • -session restart
    Description: Restarts the current session.
    Usage: -session restart
  • -svote
    Description: Sends the SSU Vote Embed you provided in setup
    Usage: -svote
  • -strike
    Description: This is used to punish staff members with major offences.
    Usage: -strike [user] (reason)
  • -warn
    Description: This is used to punish staff members with minor offences.
    Usage: -warn [user] (reason)
  • -demote
    Description: Demote someone to certain rank, removes all ranks above role provided.
    Usage: -demote [user] [role] (reason)
  • -promote
    Description: Promote someone to a new rank.
    Usage: -promote [user] [role] (reason)
  • -delete_strike
    Description: Deletes a strike from a user.
    Usage: -delete_strike [user] [strike id]
  • -delete_warn
    Description: Deletes a warn from a user.
    Usage: -delete_warn [user] [warn id]
  • -search_strikes
    Description: Searches all the strikes from the user you provided.
    Usage: -search_strikes [user]
  • -search_warns
    Description: Searches all the warns from the user you provided.
    Usage: -search_warns [user]
  • -search_promos
    Description: Searches all the promotions from the user you provided.
    Usage: -search_promos [user]
  • -search_demos
    Description: Searches all the demotions from the user you provided.
    Usage: -search_demos [user]
  • -terminate
    Description: This will terminate a user from staff.
    Usage: -terminate [user] [reason]
  • -mod
    Description: This will ping the moderator roles you provded in the setup.
    Usage: -mod
  • -staff
    Description: This will ping the staff roles you provded in the setup.
    Usage: -staff
  • -ad
    Description: This will send your advertisement that you provided in the setup.
    Usage: -ad
  • -manage_replies
    Description: Here you can add, remove, and list all auto-replies.
    Usage: -manage_replies
  • -say
    Description: This will send a message to the current channel as LCU with a given message.
    Usage: -say [message]
  • -embed
    Description: This will send a embed to the current channel as LCU with a given message.
    Usage: -embed [message]
  • -on
    Description: This will start your :m command reminders.
    Usage: -on
  • -off
    Description: This will stop your :m command reminders.
    Usage: -off
  • -config
    Description: This will allow you to reconfigure specific features within LCU.
    Usage: -config
  • -settings
    Description: This will allow you to change some settings of the bot.
    Usage: -settings
  • -info
    Description: This will allow you to get specific information about LCU.
    Usage: -settings
  • -server_blacklist
    Description: This will send a blacklist message to your current channel.
    Usage: -server_blacklist
  • -loa request
    Description: This will send a request for a LOA.
    Usage: -loa request Xy|Xm|Xw|Xd reason
  • -loa manage
    Description: This will send your current LOA which you can end.
    Usage: -loa manage
  • -loa search
    Description: This takes an optional user and will search active LOAs.
    Usage: -loa search (user)
